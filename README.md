# Bridge Crane Control System

This project develops and analyzes a bridge crane control system for industrial load transportation applications.

## Objectives

- Develop nonlinear state-space equations
- Linearize the system around equilibrium
- Analyze transfer functions and stability
- Design proportional and LQR controllers
- Implement Kalman filter state estimation
- Compare linear and nonlinear system responses

## Tools Used

- MATLAB
- Simulink
- Modern Control Theory
- LQR Control
- Kalman Filtering

## Topics

- State-Space Modeling
- Stability Analysis
- Controllability
- Observability
- Optimal Control
- Observer Design
